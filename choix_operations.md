- Demande a l'utilisateur si il va bien
- Donne le choix entre check le solde / faire un depot / faire un retrait
- Si depot :

  - Depose le montant
  - Dit aurevoir
- Retrait :
  - Demander le montant a retirier
  - Retire le montant
  - Dit aurevoir
